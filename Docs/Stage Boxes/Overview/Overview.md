# Naming Scheme
We need a comprehensive naming scheme for stage boxes and their components
## Ex\. FlexRACK\-14\-v0\.1\-M
- FlexRACK
	- Name of product
- 14
	- Number of Modular Rack slots
- v0\.1
	- hardware revision
- M
	- Variant
