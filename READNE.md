## What is MixerOS?

---

MixerOS is a open source software mixing console that leverages OpenCL to process audio samples. This allows for more efficient processors such as GPUs, FPGAs and DSP chips to process audio data in realtime. Along with a module based processing architecture gives the user the freedom to develop their unique sound.

## Features

---

- **Flexible:** Choose your own hardware to fit your needs
- **High Channel count:** Channel count is dependant on hardware so technically you could install it on a HGX B300 for more channels.
- **Community Driven:** Your input matters and we take your input to ensure that we are producing the best software possible.
- **Linux Based:** The project is based on ubuntu linux but could run on almost any distro
- **Free:** Can you really be that
